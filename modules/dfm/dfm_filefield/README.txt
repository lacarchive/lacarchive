Drupella FM for File Field
====================================


DESCRIPTION
-----------
Allows users to select files from Drupella FM for file fields.
FTP uploaded files are automatically registered and made usable in file fields.


REQUIREMENTS
----------
1. Drupella FM
2. Filefield module


CONFIGURING AND USING
---------------------
1. Go to your file field's settings page.
2. Check the box saying "Allow users to select files from Drupella FM for this field."
3. Only the roles with a Drupella FM configuration profile can use this feature.
